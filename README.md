# Espacio Romero - Alojamiento & Arte

Sitio web de **Espacio Romero** con estructura multipágina y secciones separadas.

## Estructura

- `index.html`: portada y acceso a cada sección.
- `alojamiento.html`: reservas con validación de fechas y disponibilidad.
- `artista.html`: historia y contexto artístico.
- `tienda.html`: obras y modal de catálogo.
- `resenas.html`: reseñas de visitantes con calificación por estrellas.
- `contacto.html`: canales de contacto directo.
- `styles.css`: estilos globales compartidos.
- `app.js`: lógica compartida (navegación, reservas, reseñas, tienda, efectos).

## Tecnologías

- HTML5
- Tailwind CSS (CDN)
- Font Awesome
- Google Fonts
- JavaScript Vanilla

## Cómo usar

1. Abrí `index.html` en tu navegador.
2. Navegá entre páginas con el menú superior.
3. Asegurate de tener las imágenes `foto1.jpeg`, `foto2.jpeg`, `foto3.jpeg` y `foto4.jpeg` en la carpeta `img/` del proyecto.
